# Apple Style Product Page

A premium Apple-inspired product landing page built using HTML and CSS.

The project focuses on clean layouts, modern typography, responsive design, and product-focused user experience.

## 🚀 Features

- Apple Inspired Design
- Premium Hero Section
- Product Showcase
- Feature Cards
- Specifications Section
- Customer Reviews
- Pricing Section
- Responsive Layout
- Smooth Hover Effects

## 🛠️ Technologies Used

- HTML5
- CSS3
- Flexbox
- CSS Grid
